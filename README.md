CuraPluginOven
==============

Build steps:
------------
* git clone
* git submodules
* compile_all
* copy tree without .files and *.py files
* zip it!

File formats:
-------------
* .api-4.curaplugin - Cura 3.2 and downwards
* .sdk-5.curapackage - Cura 3.3 - Cura 3.6
* .sdk-600.curapackage - Cura 4.0.0 - ?
